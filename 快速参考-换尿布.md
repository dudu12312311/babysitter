# 🍼 换尿布任务 - 快速参考卡

## 🚀 一键启动

```bash
# 双击运行
start_app.bat

# 或命令行
python main.py
```

## 🌐 访问地址

```
http://localhost:5000/diaper
```

---

## 🎮 操作流程

```
1. 打开网页 → 看到哭脸 😭
2. 调整参数 → 提腿速度、擦拭次数、放置顺序
3. 点击按钮 → "开始换尿布"
4. 观察结果 → 成功显示笑脸 😊，失败继续哭脸 😭
5. 重新开始 → "再来一次"
```

---

## ⚙️ 推荐参数

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 提腿速度 | 3-4秒 | 太慢会触发喷射袭击 |
| 擦拭彻底度 | 7-9次 | 太少不干净 |
| 放置顺序 | 正确 | 先擦拭再穿新尿布 |

## 📊 数值变化

**成功时：**
- 清洁度：+20 ~ +40
- 快乐度：+10 ~ +20
- 健康值：+5 ~ +10
- 压力值：-5 ~ -15

**失败时：**
- 清洁度：-10 ~ 0
- 快乐度：-5 ~ -15
- 健康值：-5 ~ 0
- 压力值：+10 ~ +20

---

## ⚠️ 注意事项

- 提腿速度 > 5秒 → 30% 概率触发"喷射袭击"
- 擦拭次数 < 5次 → 清洁度不足
- 错误顺序 → 任务失败

## 🔧 故障排除

**应用无法启动？**
```bash
python -m pip install Flask==2.3.3
```

**端口被占用？**
```bash
# 修改 main.py 最后一行
port = int(os.environ.get('PORT', 8000))
```

**导入错误？**
```bash
python test_setup.py
```
