# 硬核育儿模拟器 - 游戏演示使用说明

## 🎮 运行方式

```bash
python game_demo.py
```

## ✨ 新增功能

### 1. 用户交互输入
每个任务都支持用户输入参数，体验真实游戏玩法：

- **直接回车** = 使用默认值（快速演示）
- **输入数值** = 自定义参数（深度体验）

### 2. 阶段选择
启动时可以选择要演示的阶段：
1. 0-3月新生儿阶段
2. 3-12月婴儿阶段  
3. 1-2岁幼儿阶段
4. 2-3岁学龄前阶段
5. 困难模式演示
6. 性格对比演示
7. 随机事件演示
8. 全部演示

## 💡 使用技巧

### 快速演示模式
所有输入直接按回车，使用默认值：
```bash
python game_demo.py
# 然后一路回车
```

### 深度体验模式
每个任务都输入自定义参数，体验不同结果：
```bash
python game_demo.py
# 根据提示输入不同的数值和选项
```

### 对比测试
运行多次，尝试不同参数组合：
- 水温过高 vs 水温适中
- 快速反应 vs 慢速反应
- 天使宝宝 vs 高敏宝宝

## 🎯 游戏机制体验

### 1. 参数影响结果
- 水温 37-40°C = 完美
- 水温 > 40°C = 烫伤风险
- 水温 < 37°C = 太凉

### 2. 性格差异
- 天使宝宝：对胡萝卜友好
- 高敏宝宝：对胡萝卜过敏

### 3. 技能判定
- 反应时间 < 2秒 = 成功
- 反应时间 > 2秒 = 失败

### 4. 状态变化
每次任务后查看：
- ❤️ 健康值
- 🍼 饥饿度
- 🛁 清洁度
- 😊 快乐度
- 💕 亲密度

## 🐛 故障排除

### 输入错误
- 输入非数字 → 提示重新输入
- 超出范围 → 提示重新输入
- 按 Ctrl+C → 取消操作

### 默认值
所有输入都支持直接回车使用默认值，不会卡住
